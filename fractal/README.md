# Tree Fractal

This is a very simple implementation of a fractal tree.<br />

## Specifications

Python version: `3.6.2` <br />

## Usage

1. Download this folder and store it in your computer.
2. Go to the folder's directory.
3. Type on Powershell: `python tree.py`.
